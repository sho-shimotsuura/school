# README  
##　開発言語  
- Ruby3.0.1  
- Rails6.1.6  
## 就職termの技術  
- devise  
- rails_admin  
- お気に入り機能  
- メッセージ機能  
## カリキュラム外の技術  
- ransack  
- Active Storage  
- geocoder  
## 実行手順  
$git clone git@github.com:sho-shimotsuura/school.git  
$cd school  
$bundle install  
$rails db create && rails db:migrate  
$rails s
